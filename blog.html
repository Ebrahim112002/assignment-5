Question-1: What are the different ways to select an element in the DOM?

There are several ways to select elements in the DOM:
getElementById(): Selects an element by its unique id. getElementsByClassName(): Selects elements by their class name. querySelector(): Selects the first element matching a CSS selector. querySelectorAll(): Selects all elements matching a CSS selector.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML: Returns or sets the HTML content inside an element, including tags.
innerText: Returns or sets the visible text content, respecting CSS styling.
textContent: Returns or sets the raw text content, ignoring any HTML tags or styling.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where an event listener is attached to a parent element instead of individual child elements. It uses event bubbling to catch events from child elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is the process where an event triggered on an element propagates upwards to its parent elements, allowing parent elements to handle the event.

Question-5: How do you create, add, and remove elements using JavaScript?

Create: Elements are created using document.createElement().
Add: Elements are added to the DOM using methods like appendChild() or insertBefore().
Remove: Elements are removed from the DOM using removeChild() or remove().
